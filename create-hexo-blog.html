    hexo

    Hexo 是一個快速、簡單且強大的網誌框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在幾秒鐘內透過漂亮的主題產生靜態檔案。

    安裝需求

    • Node.js (Node.js 版本需不低於 8.6，建議使用 Node.js 10.0 及以上版本)
    • Git

    安裝Hexo

    接下來透過npm安裝Hexo

    安裝完後查看Hexo版本

    接下來就可以開始初始化部落格
